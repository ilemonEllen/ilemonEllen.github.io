Windows Terminal配置

Windows Terminal](https://github.com/microsoft/terminal)是一款集成了Windows上的PowerShell、CMD以及Windows Linux子系统的终端。Windows自身的命令控制台过于简陋，所以我们不妨使用Windows Terminal来替代cmd和power shell，只需要简单的配置一些功能，就能实现很多常用的Linux操作命令。过程如下： Vim配置 Vim是一款优秀的Linux文本编辑器。我们在Windows下编辑一些一些配置文件的时候很有作用。 1.首先，我建议先去下载一个Vim,安装的时候注意路径中不要包含中文路径。 2.然后，配置环境变量。找到Vim.exe的二进制文件位置，例如我的Vim安装位置在C:\Program Files (x86)\Vim\vim82,然后到设置——系统——关于——高级系统设置——环境变量——path——编辑——新建。然后将Vim.exe的路径复制粘贴上去。（例如我的就是C:\Program Files (x86)\Vim\vim82）。然后重新打开Windows Terminal终端输入vim -h检验环境变量是否配置成功。 Git Bash配置 GitBash是一个用于MicrosoftWindows环境的应用程序，被集成在git下面。简明扼要的说一句就是能够实现很多Linux命令操作。更多资料——https://zh.wikipedia.org/zh-cn/Git 其实，我自己在网上看到很多的将Git Bash配置到Windows Terminal下都是通过修改Windows Terminal中的JSON文件来实现的。但是实际中，Windows Terminal已经精简了很多的冗长的代码，添加配置文件也变得更加的简单。 1.首先，你需要去下载一个Git 或者是Git for Windows,两者区别我自己目前使用的是Git。 2.打开Windows Terminal终端，进入设置——添加配置文件。填写名称，选择文件路径（找到自己git bash.exe的二进制文件路径）例如我的git 的bash.exe在D:\BigDataTools\Git\Git\bin下。你可以添加一个图标（当然也可以不要）。之后重启终端启用git bash输入git –version看看能否得到git版本，可以的话就代表成功了。 但是，完成相关的bash配置之后我们会发现，当我们使用命令浏览磁盘下具有中文的文件夹时会出现乱码现象。其实，这个很好解决，我们只需要在Git文件路径下找到bash.bashrc文件，例如我的路径D:\BigDataTools\Git\Git\etc。 然后打开文件进行编辑，在文件后面添加 export LANG="zh_CN.UTF-8" export LC_ALL="zh_CN.UTF-8" 之后，保存退出，重启终端，这个时候我们就会发现问题解决了。 其实将Git Bash集成到Windows Terminal中之后，完全也可以使用Vim，不需要下载添加VIM。当然，你甚至可以选择一个cmder直接下载使用，内置git bash，无需任何的配置，省时省力。 参考博客：https://blog.csdn.net/jun8148/article/details/97245249 Bash图标:

浏览器上网的日常防护

例如，你在一个网站上看到一个商品，点击了解了相关的信息，但是并没有购买，那么第二天当你打开类似的网站的时候，你就看到相关的广告推荐。这种现象并非偶然，而是通过一系列的技术手段实现的。作为个人我们能当然不希望自己在网络上的行为活动被一些幕后的个人和公司监视。所以我们需要了解相关的追踪方法然后采取相对应的措施。 什么是浏览器指纹——追踪技术 “浏览器指纹”是一种通过浏览器对网站可见的配置和设置信息来跟踪Web浏览器的方法。这些信息包括：你的系统信息，你使用的浏览器型号，你IP所在的位置等等。而当你访问一个网站的时候，这个网站往往会弹出提示——是否要接受我们使用cookie。现在描绘浏览器用户画像主要是通过Cookie,webRTC等进行追踪定位用户。相关专业性的理论不需要了解太多，你只需要知道你每浏览一个网站，网站都会在你磁盘成生成“临时文件”（暂且把cookie称之为临时文件，虽然不严谨，但是好理解），而网站的运营者可以通过“临时文件”来知道你上次来这个网站是什么时候，你还去过哪些网站，之后各种各样的“猜你喜欢”就来了。 Windows中的使用技巧

掌握一些Windows的实用小技巧，能更好优雅的使用Windows系统，同时也能让你的Windows使用起来更加的顺手。以下是个人总结的一些使用技巧。 参数说明：Windows10pro，X64，20H2。 清理系统盘 一般系统默认都是C盘，C盘中如果有过多的文件导致系统盘空间不足的话就会拖慢系统的运行速度。所以需要保持系统盘的健康。 打开资源管理器，找到此电脑下的系统盘，单击右键选择属性，然后点击磁盘清理，再点击清理系统文件，勾选你想要清理的系统文件即可（一般来说勾选临时文件不会有什么影响的）。 打开Windows的设置，依次点击系统-存储-打开存储感知配置，当然你可以进一步设置系统自动清理的间隔时间。 搜索框中搜索“运行”打开，里面输入temp，然后回车（或者你可以进入系统盘-Windows-temp）。然后将temp文件夹下的所有临时文件删除(shift+delete) 。再次打开“运行”输入%temp%回车（或者你可以在C:\Users\PC用户名\AppData\Local\Temp）下面找到这些文件，然后按下shift+delete. 定时清空回收站：无论你是直接右键删除文件，还是上面的shift+delete删除，这都不能将文件彻底的删除掉。系统还是会保留还原点在回收站中，而回收 站里的文件都将保留在系统盘上。所以定时清空你的回收站是很有必要的一件事情。 定时清除浏览器中的缓存和cookie。浏览器一般都是默认安装在系统盘中，所以你在浏览网页时候产生的缓存就会保存在你的系统盘下面，特别是对于cookie 的清理，定时的清理也能够保护自己的隐私。 借助第三方的清理工具：你可能觉得单纯的系统措施不能达到你预想的效果，或者是觉得操作过于繁琐，那么你可以试试借助第三方的工具来实现清理。 Dism++:这是一款很实用的工具，你也可以用它来清理你不想要的文件和缓存。当然，如果你也有有一些偏执的话，你不妨再下载一个**Bleach bit**专门用来清理浏览器缓存。不过，卸载也是其中重要的一个组成部分，单纯的Windows卸载面板是没有办法将一个软件彻底卸载干净的，里面常常还会有注册表的残留。所以，采用专业的卸载工具就显得极为重要——**geek uninstaller**是一个比较不错的选择。 关闭不必要的自启动选项 过多的自启动选项是会拖慢系统的开机速度的。所以我们需要对一些自启动选项实行禁用。ctrl+shift+esc打开任务管理器找到上面的启动项，然后将不必要的启动项右键禁用掉即可。当然你也可以在设置-应用-启动项中找到一些自启动的应用，将对系统影响较小的应用或者程序关闭掉即可。 清除电脑上的恶意广告程序 搜索框中搜索cmd打开命令控制符在里面输入mrt，然后回车就会弹出一个Windows内置的清除程序，选择一个扫描模式，等待扫描完成,点击清除就可以。不过，这种办法只适用于一些小儿科式的恶意程序，真正的恶意程序还是需要借助专业的杀毒工具来处理，除此之外软件下载的渠道的选择和软件本身的选择（有些软件本身就是流氓性质，例如金山毒霸和腾讯电脑管家之类的）也是十分的重要。 修复系统盘文件 如果你不小心删除了系统盘中比较重要的文件，但是自己又不知道具体是哪些。那么你可以使用系统文件检查器工具修复丢失或损坏的文件。 使用管理员权限打开命令提示符：输入Dism /Online /Cleanup-Image /ScanHealth（扫描全部系统文件和系统映像文件是否与官方版一致） 等待扫描完成：输入Dism /Online /Cleanup-Image /CheckHealth（这里检查文件的损坏程度，结果一般有两种：一般性损坏可以修复和情况严重完全无法修复） 如果损坏可以被修复的话，直接输入DISM /Online /Cleanup-image /RestoreHealth（这里是联网将系统映像文件中与官方文件不相同的文件还原成官方系统的源文件，但是系统中安装的软件和用户设置保留不变） 最后只需要耐心等待一段时间就可以了。更多的详情可以参考官方给出的答案：https://bit.ly/3wwdc3U 显示文件扩展名和隐藏文件 有些时候你在你的资源管理器当中可能无法看到文件的扩展名或者是某些隐藏的文件。这个时候你就需要打开资源管理器-点击上面查看-在隐藏文件和展示文件扩展名前面打勾即可。 启用Windows高级搜索 Windows的文件搜索是一个比较鸡肋的方面，你可以通过进入设置-搜索-搜索Windows中的经典改为高级。当然你也可以使用第三方的搜索工具，例如**everything**,**listary**等来提高你的文件搜索效率。 开启使用bitlocker 如果你的系统是Windows专业版，那么强烈建议开启bitlocker来保护你的系统盘远离恶意的脚本和程序。你只需要在控制面板中找到bitlocker然后将其打开就可以了（需要保护好自己的bitlocker密钥文件，否则里面的文件将再也无法打开）。 优雅的使用Microsoft拼音输入法 我们进入设置-时间和语言-语言-语言偏好-点击简体中文-点击更多选择-在键盘中找到Microsoft拼音-点击更多选择，然后就可以对微软拼音输入进行个性化的设置，强烈建议在高级选项中打开U模式输入和V模式输入，前者可以使用U加上陌生字的一部分来实现查字功能，后者主要用来快速输入时间，日期等信息。你甚至可以词典中加入你想要的备用句子。 光标醒目 如果你想要自己鼠标的指针更加的醒目，那么你可以在设置-轻松使用-指针里面找到指针这一项，然后可以根据自己的需要来对光标进行大小和颜色的调整。 快捷键使用 如果你是一个习惯使用快捷键进行操作的人，那么你大概率可能需要对某个应用进行快捷键操作。这个时候你可以在桌面上找到你想要添加快捷键启动的应用快捷方式，然后选中右击-选择属性-点击快捷方式在下面的快捷键那一栏中按下你想要设置的快捷键，然后应用，确定就可以了。...

罗翔老师语录

罗翔老师在我看来不仅仅给人上了生动形象的刑法课，更重要的是将法律的精神传递给普罗大众，让更多的人知道法律到底是为了什么而存在的，法律和人属性之间的千丝万缕关系。看他视频时间长了之后你就发现，罗翔老师对于人生的思考也是极为深刻的，往往是出口成章，发人深省。身为一名富有哲学智慧和广博学识但现在一直时刻提醒自己保持谦卑的老师，我真的很佩服罗翔老师的为人做派。所以我打算在自己的一篇博客上面记录一些罗翔老师的经典语录，这里既有他本人自己的陈述回答也有他引用的名人名言。 1.什么叫做好？ 这个世界上有太多关于好的标准了， good，如果加个s，就变成了goods,那就变成了商品。 所以做一切事情最重要的就是赚钱吗？ 什么叫好医生，赚钱就是好医生？ 什么是好老师，赚钱就是好老师？ 什么是好记者，赚钱就是好记者？ 什么是好演员，赚钱就是好演员？ 那这样做一切职业分工没有意义了， 你要去寻找什么叫good，然后你才可以去过good的一生。 你才可以避免去随波逐流，你才内心会有一种笃定， 你才会拒绝去和别人攀比，也拒绝被别人所攀比。 ——罗翔《怎么过好自己的每一天》 2.我怕滥用这种影响力，我怕我驾驭不了这种影响力，我更怕迷恋这种影响力。 3.真正的知识，要从书本走向现实，真正的法律，并不仅仅是抽象的逻辑，而是每一个人鲜活的故事。要让公平和正义，不仅仅要在书上得到体现，更重要的，是在每个个案中得到回响。 4.所谓的教育，就是把一个人的内心，真正引导出来，帮助他成长成自己的样子。 5.有朋友让我谈谈虚伪，我说自己就很虚伪。但惟有人会虚伪，动物不会。当你意识到自己的虚伪可能意味着你愿意朝着更高的标杆去前进。虽然这种前进反而会让你更深刻的认识到虚伪的本性，不断地督促你正视自己内心深处的幽暗。这种幽暗翻江倒海，稍有不慎就会外泄成灾，所以需要去倾听他人的批评和自己内心的责问，不做狡辩，日拱一卒，每天离虚伪远一点。共勉。 我浏览器中那些不可或缺的扩展

我其实自己早在GitHub上的文章中就有写到插件对于浏览器的重要性，合理的使用一些插件能让你的浏览器如虎添翼。在这里我也打算说说自己浏览器中那些不可或缺的插件神器，到时候有人再问我使用的什么插件的时候，我就直接把这个博客给他看。 ADguard 这是一个广告过滤插件，能够帮你过滤掉网站上你不想看到的广告，当然，也可以完美过滤YouTube视频中的广告（本人亲测有效）。广告过滤插件那么多，有这一个就够了。 Dark Reader 这是一个“护眼插件”，通过这个插件你可以轻松的将大部分网页渲染成黑色，以此来达到降低夜间白色背景光对眼睛的刺激。 bitwarden 这是一款开源的密码管理软件。开源的密码软件不在少数，但是这个上手就比较简单。 privacy Badger 这是一个自动学习并且能够阻止网页上不可见追踪器的插件。 Randrom User-Agent 自动随机的改变用户的浏览器头部信息，减少被刻画用户画像的风险。 Pocket 将自己想要稍后阅读或者是自己喜欢的文章，网页收藏起来。算是一个收集箱。 PopupOFF 阻止顽固的网页弹窗和多余的页面边缘元素，提升阅读和浏览体验。 Clearurls 一款开源插件，去除url中的跟踪元素，保护个人隐私。 Tampermonkey 浏览器中功能最强大的插件，能够配合**Greasy fork**这个脚本网站实现各种各样的功能。 Disconnet 能够提升网页的响应速度，阻止网页追踪器，保护上网安全。 IDM 这是一个多线程的下载插件（需要付费），能够自动嗅探资源，断点恢复下载，多线程 高速下载，下载大文件用它就对了。 WebRTC Leak Prevent 禁用自己浏览器上的webRTC，保护自己的真实IP不泄露。 https://ip.voidsec.com/ 一个用来检测webRTC有没有泄露你真实IP的网站，没有泄露则webRTC处没有IP显示。 网站管理员可以轻易地通过WebRTC看到用户的真实IP地址，即使用户使用VPN来隐藏自己的IP，也可以被追踪到。 解决方法： Mozilla Firefox：在地址栏输入 “about:config”，搜“media.peerconnection.enabled” 并双击将值改为 “false”，WebRTC 将被关闭。 Google Chrome：在谷歌应用商店安装谷歌官方扩展 “WebRTC Leak Prevent”，在插件选项里找到 “IP handling policy:” 选择第三项 “Disable non-proxied UDP”，并点击下方 “Apply settings”。只要是采用chromium内核的浏览器都可以使用此方法。 扩展和插件区别 其实有很长一段时间我自己也将插件和扩展混为一谈，但其实两者是两个完全不同的概念。 插件(plugins)：　在功能上，插件通常是用来渲染页面里的 或 标签。 插件通常用操作系统的本地代码（也叫“原生代码”）编写，可以调用操作系统的 API。 形式上，插件以动态库（Windows 上就是 DLL 文件）的方式，加载到浏览器的进程内。由于使用本地代码编写， 插件通常依赖于特定的操作系统（不同系统的插件不能混用）。...
